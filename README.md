## 使用方法

安装依赖（目前只有 requests

```shell
sudo -H pip3 install -r requirements.txt
```

```shell
python3 main.py yourNumber yourpassword
```

RasberryPi 文件夹下面是在树莓派下运行的 python 代码以及启动脚本，自动登陆并将 ip 发送到指定邮箱
